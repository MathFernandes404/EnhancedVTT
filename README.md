# Enhanced VTT ✦ Extension for Roll20

Uma extensão avançada para o navegador (Chrome, Brave, Edge) desenvolvida para aprimorar o gerenciamento de fichas de personagens e tokens no Roll20 (exclusivamente para a ficha clássica **D&D 5e (2014/OGL)**).

## 🚀 Principais Recursos
- **Sincronização de Imagens**: Altere imagens e avatares de fichas e panfletos rapidamente a partir de URLs.
- **Importação/Exportação Avançada**: Exporte as fichas de personagens em JSON e importe-as novamente.
- **Evita Duplicados**: Importação com sincronização diferencial que substitui itens alterados e remove os deletados (ex: magias e habilidades), sincronizando com Firebase.
- **Configurações do Token**: Modifique dimensões, auras de combate, e vincule as três barras de atributos a variáveis da ficha.
- **Galeria de Tokens**: Salve configurações específicas de tokens (ex: montaria, combate, furtivo) na galeria local.
- **Design Moderno e Compacto**: Interface minimalista com tema escuro e suporte para colapso rápido (dê dois cliques na barra superior para minimizar/expandir).

## 📦 Como Instalar (Carregar Unpacked)
Como esta é uma extensão customizada e não está publicada na Chrome Web Store:
1. Faça o download da última release em `.zip` e extraia os arquivos no seu computador.
2. Abra o navegador e vá para a página de extensões: `chrome://extensions/`.
3. Ative o **Modo do Desenvolvedor** no canto superior direito.
4. Clique no botão **Carregar sem compactação** (Load unpacked) no canto superior esquerdo.
5. Selecione a pasta onde a extensão foi extraída (a pasta contendo o arquivo `manifest.json`).
